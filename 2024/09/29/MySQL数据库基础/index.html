MySQL概述

数据库相关概念

  • 数据库：数据存储的仓库
  • 数据库管理系统：操纵和管理数据库的大型软件
  • SQL：操作关系型数据库的编程语言，是一套标准

  • 数据模型

  • 关系型数据库（RDBMS）

    • 概念：建立在关系模型基础上，由多张相互连接的二维表组成的数据库
    • 特点：
      1. 使用表存储数据，格式统一，便于维护
      2. 使用SQL语言操作，标准统一，使用方便

SQL

SQL通用语法

  • SQL语句可以单行或多行书写，以分号结尾
  • SQL语句可以使用空格/缩进来增强语句的可读性
  • MySQL数据库的SQL语句不区分大小写，关键字建议使用大写
  • 注释：
    • 单行注释：-- 注释内容 或 #注释内容（MySQL特有）
    • 多行注释 /* 注释内容 */
