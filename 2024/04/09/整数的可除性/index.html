整数的可除性

1.整除的概念&带余除法

定义1： 设 \(\forall a, b \in \mathbb{Z}\), 其中 \(b \neq 0\), 若 \(\exists q \in \mathbb{Z}\) 使得 \(a = b \cdot q\), 则称 \(b\) 整除 \(a\) 或者 \(a\) 能被 \(b\) 整除，数学符号记作 \(b \mid a\), 此时我们把 \(b\) 叫做 \(a\) 的因数, \(a\) 叫做 \(b\) 的倍数。若 \(\nexists q \in \mathbb{Z}\) 使得上述等式成立，则说明 \(b\) 不能乘除 \(a\) 或者 \(a\) 不能被 \(b\) 乘除，数学符号记作 \(b \nmid a\)

定理1（传递性）： 若 \(\exists a, b, c \in \mathbb{Z}\) 且 \(b \mid a , c \mid b\), 则 \(c \mid a\)

证明： 由 \(b \mid a, c \mid b\) 知，\(\exists q_1,q_2 \in \mathbb{Z}\) 使得 \(a = q_1b, b = q_2c\), 所以 \(a = q_1q_2c\), 不妨令 \(q = q_1q_2\) 则有 \(q \in \mathbb{Z}\) 使得 \(a = qc\), 即 \(c \mid a\)

定理2： 若 \(m \mid a\) 且 \(m \mid b\) 则有 \(m \mid (a \pm b)\)

证明： 由 \(m \mid a, m \mid b\) 知 \(\exists q_1, q_2 \in \mathbb{Z}\) 使得 \(a = q_1m, b = q_2m\), 则 \(a \pm b = (q_1 \pm q_2)m\), 令 \(q = q_1 + q_2\) 则有 \(a \pm b = qm\), 即 \(m \mid (a \pm b)\)

定理3： 若 \(a_1, a_2, ..., a_n\) 皆为 \(m\) 的倍数，\(q_1, q_2, ..., q_n\) 为任意的 \(n\) 个整数，则 \(q_1a_1 + q_2a_2 + ... + q_na_n\) 仍为 \(m\) 的倍数

证明： 可知 \(\exists c_1,c_2,...,c_n \in \mathbb{Z}\) 使得 \(a_1 = c_1m,a_2 = c_2m,...,a_n = c_nm\), 所以 \(q_1a_1 + q_2a_2 + ... + q_na_n = (q_1c_1 + q_2c_2 + ... + q_nc_n)m\), 令 \(q = (q_1c_1 + q_2c_2 + ... + q_nc_n)\) 则有 \(q_1a_1 + q_2a_2 + ... + q_na_n = qm\), 即 $ m (q_1a_1 + q_2a_2 + ... + q_na_n)$
